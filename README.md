# Dance Sync ESP32 项目说明 (S3 双手版 Ver-D)

这是一个基于 **两块 ESP32-S3** 和 **两块 MPU6050** 六轴传感器的体感舞蹈同步系统。通过 BLE（蓝牙低功耗）将传感器数据发送至浏览器前端，实现双手音乐实时互动和视觉反馈。

## 硬件清单
- ESP32-S3 开发板 × 2
- MPU6050 六轴传感器 × 2
- 面包板 × 2、跳线若干、Type-C 数据线 × 2

## 连线方案（两块板完全一致）

| MPU6050 引脚 | ESP32-S3 引脚 | 说明 |
| :--- | :--- | :--- |
| **VCC** | **3V3** | 电源（红灯应亮起） |
| **GND** | **GND** | 地线 |
| **SCL** | **GPIO 5** | I2C 时钟线 |
| **SDA** | **GPIO 4** | I2C 数据线 |
| **AD0** | **GND** | **必须接地**，锁定芯片地址至 0x68 |

## 软件驱动说明
采用 **底层 Wire 直接读取** 策略，绕过 Adafruit 官方库对芯片 ID 的校验限制。

## 两块板的唯一区别
| 角色 | 代码文件夹 | 蓝牙名称 |
| :--- | :--- | :--- |
| **主手** | `esp32_ble_sensor/` | `ESP32` |
| **副手** | `esp32_ble_sensor_mini/` | `ESP32-mini` |

## 如何运行
1. 分别上传两个 .ino 文件（IDE: ESP32S3 Dev Module, USB CDC: Disabled）
2. 串口看到 `蓝牙已开启` 即成功
3. 浏览器打开 **dance_sync_verD.html**（Ver-D 新版前端）
4. 点击启动音频引擎 → CONNECT 连接双手 → 起舞！

## Ver-D 前端改进
- **UI 重构**: 删除 WebSocket，主手/副手参数嵌入各自卡片
- **沉浸模式**: 隐藏面板，左右两侧浮动切换按钮
- **BLAST 爆破**: 替换 SFX 白噪音，低沉炸弹效果
- **副手摄像机**: Flight Sim 中副手控制俯仰角
- **景深优化**: 远处星球暗淡、近处明亮
- **默认线粗**: Joy Division 加粗至 1.4，Topo 降至 0.8
